# 🧩 Ejercicios y Retos Prácticos

Esta sección contiene ejercicios organizados por nivel de dificultad para practicar los conceptos del curso.

## 📄 Documentos disponibles

> Coloca aquí tus archivos `.md` con ejercicios. Algunos temas sugeridos:

| Archivo | Nivel | Tema |
|--------|-------|------|
| `ejercicios_dart_basico.md` | 🟢 Básico | Variables, funciones, colecciones en Dart |
| `ejercicios_dart_avanzado.md` | 🟡 Intermedio | Clases, async/await, generics |
| `ejercicios_flutter_widgets.md` | 🟢 Básico | Construir layouts con widgets básicos |
| `ejercicios_flutter_estado.md` | 🟡 Intermedio | Gestión de estado con `setState` y `Provider` |
| `reto_feature_completa.md` | 🔴 Avanzado | Crear una feature completa con la CLI y conectarla a una API |
| `reto_proyecto_final.md` | 🔴 Avanzado | Proyecto integrador aplicando todo lo aprendido |
