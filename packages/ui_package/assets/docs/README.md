## 🚀 ¿Por dónde empezar?

1. Si eres nuevo en Dart → comienza en [`01_dart/`](./01_dart/README.md)
2. Si ya sabes Dart y quieres aprender Flutter → ve a [`02_flutter/`](./02_flutter/README.md)
3. Para conocer los patrones y templates del proyecto → visita [`03_patrones_y_templates/`](./03_patrones_y_templates/README.md)
4. Para usar la herramienta CLI del monorepo → revisa [`04_cli/`](./04_cli/README.md)
5. Para practicar con ejercicios → entra a [`05_ejercicios/`](./05_ejercicios/README.md)
